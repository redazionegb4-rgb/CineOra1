# CineOra – Build 1

App iOS SwiftUI per film attualmente al cinema, prossime uscite, categorie, ricerca e preferiti.

## Requisiti
- Xcode 15 o successivo
- iOS 17 o successivo

Aprire `CineOra.xcodeproj`, selezionare il proprio Team in Signing & Capabilities e avviare.

La chiave TMDB v3 è già configurata in `TMDBService.swift`. Per una pubblicazione commerciale, verificare la licenza TMDB.
